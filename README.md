## How to use it !

git clone https://github.com/futureunicorne/Mobeye.git

cd Mobeye

if you  use 'yarn' :

    - yarn

    - yarn run data_build (No need to use it, data.json already exist)

    - yarn start

if you  use 'npm' :

    - npm install

    - npm run data_build (No need to use it, data.json already exist)

    - npm start
